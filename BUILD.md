## Building from Source

### Requirements
- SAS/C compiler 6.58
- NDK 3.2
- ToolKit

### Build Commands
```bash
# Using SMakefile
cd Source/
smake Codex
smake install ;Will copy Codex to the SDK/C drawer in the project directory
```

## Installation

1. Find the Codex executable and matching icon in SDK/C/ in this distribution
